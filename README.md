# dashboard-vehiculos
# Dashboard de Visualización de Datos

## Instrucciones de ejecución

1. Ingresar al siguiente enlace:
   https://mybinder.org/v2/gh/andrelo1427-cell/dashboard-vehiculos/HEAD

2. Esperar a que cargue el entorno (puede tardar unos minutos).

3. Abrir una terminal.

4. Ejecutar el siguiente comando:
   python app.py

5. Abrir el dashboard en el navegador agregando:
   /proxy/8050/

## Descripción

Este proyecto corresponde al desarrollo de un dashboard interactivo utilizando Python, Dash y Plotly para la visualización de datos.

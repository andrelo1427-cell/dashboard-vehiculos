# 🚗 Dashboard Análisis de Vehículos

## 📊 Descripción
Este proyecto analiza la relación entre las características técnicas de los vehículos y su precio, utilizando modelos estadísticos como regresión lineal y regresión logística.

## 🎯 Objetivo
Determinar si variables como caballos de fuerza y número de cilindros influyen significativamente en el precio de los automóviles.

## 🧠 Modelos utilizados
- Regresión lineal
- Regresión logística
- Contraste de hipótesis

## 📈 Resultados
- R² ≈ 0.50 → relación moderada
- Accuracy ≈ 0.88 → alta capacidad de clasificación
